Rails punya apa pun yang kamu butuhkan

Ruby on Rails bukan framework yang minimalis, melainkan framework metropolis. Ruby on Rails merupakan sebuah framework yang dilengkapi dengan semua hal yang dibutuhkan oleh institusi besar untuk menjalankan aplikasi yang besar dan berat seperti Basecamp, Github atau Shopify.

Hal ini berarti bahwa banyak hal yang harus dipelajari! Tapi hal ini juga berarti bahwa Kamu dapat mempelajarinya sedikit demi sedikit. Kamu tidak perlu menjadi seorang ahli dalam semua aspek di Ruby on Rails untuk memulai membuat aplikasi yang hebat.

Pola dari sebuah aplikasi Rails adalah desain Model-View-Controller yang mengatur logika pemrograman menjadi 3 lapisan utama. Model adalah tempat dimana semua logika bisnismu diletakan. Disinilah kamu menghitung apakah invoice selanjutnya sudah kadaluarsa atau berapa banyak tugas yang tersisa untuk diselesaikan dalam suatu project. Controller bertugas untuk menerima click dan tap dari user, mengarahkan perintah tersebut ke model, dan kemudian me-render template HTML atau JSON sebagai View.

Di dalam Rails, model secara umum diatur menggunakan Active Record (dengan beberapa backup menggunakan Active Model). Active Record adalah framework yang digunakan untuk berkomunikasi dengan database. Controller diatur menggunakan Action Controller dan View diatur menggunakan Action View, yang secara bersamaan disebut sebagai Action Pack.

Selain framework inti di atas, kami juga menyediakan Action Mailer, yang digunakan untuk mengirim email dalam bentuk HTML maupun plaintext. Action Cable yang digunakan untuk melakukan live update pada halaman melalui WebSocket. Active Support berguna meningkatkan bahasa Ruby, standard library, ekstensi tambahan, utility dan transversal lainnya yang digunakan untuk aplikasi web. Dan yang terakhir adalah Railties yang menyediakan semua command (perintah), generator, dan glue atau memodifikasi proses inisialisasi untuk menyatukan semua framework ini menjadi suatu sistem terpadu yaitu Ruby on Rails.
